See backend login code: 

```py
@router.post("/login")
async def admin_login(username: str = Body(...), password: str = Body(...)):
    admin = db.admins.find_one({"username": username})
    if not admin or admin.get("password") != password:
        raise HTTPException(status_code=401)
    return {"access_token": "admin"}
```
